This page shows some data on the genomic trends of the COVID-19 virus in the UK. Note that all graphs are plotted based on the date the virus sample was taken, and it takes some time for samples to be sequenced. Consequently, data for the last month or so will be based on fewer samples.

This plot shows the prevalence of various mutations which have been suspected to have an effect on the virus's behaviour:

This graph shows the prevalence of the named variant "VUI-202012/01", which has an unusually large number of mutations and is hypothesised to be more infectious .

The VUI-202012/01 variant is identified here by the presence of the mutations n501y, q27stop, del_21765_6, and p681h.
